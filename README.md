# Currency Converter & GDP Growth Analysis

## 📌 Project Overview

**Currency Converter & GDP Growth Analysis** is a web-based application that provides real-time currency conversion along with an interactive visualization of global GDP growth data.  
The project is designed to help users understand international currencies and economic performance of countries through an intuitive and visually rich interface.

This project demonstrates the use of **API integration, interactive SVG maps, and frontend web technologies** for economic data visualization.

---

## 🎯 Objectives

- Convert currencies using real-time exchange rates
- Visualize global currency and country-level economic data
- Display GDP growth information in an interactive manner
- Provide a simple and user-friendly interface for economic insights

---

## 🚀 Features

### 1. Currency Converter 💱
- Convert between **170+ international currencies**
- Uses real-time exchange rates
- Supports user input for quick and accurate conversion

### 2. Interactive World Map 🌍
- Clickable SVG-based world map
- Displays country and currency details on hover
- Enhances geographical understanding of global currencies

### 3. GDP Growth Visualization 📈
- Presents GDP-related economic data for countries
- Helps users analyze economic growth trends
- Useful for students, analysts, and economic enthusiasts

### 4. Responsive UI 🎨
- Clean and modern UI design
- Works across desktops and laptops
- Smooth animations and hover effects

## ⚙️ How It Works

1. User selects source and target currencies
2. Application fetches live exchange rates via API
3. Converted value is displayed instantly
4. Interactive world map allows users to explore:
   - Country
   - Currency
   - Economic context (GDP growth data)

---

## ▶️ How to Run the Project

### Prerequisites
- A modern web browser (Chrome, Edge, Firefox)

### Steps
1. Download or clone the project repository
2. Open `index.html` in a browser
3. Navigate to the **Currency Converter** section
4. Start converting currencies and exploring GDP data

> No backend or server setup required.
